Programming Languages in Rocq

A machine-checked port of Programming Languages in Haskell — each chapter’s language, interpreter, and metatheory, formalized and proved in the Rocq prover by Perry Alexander with extensive help from Claude.

Every chapter has a lecture (the development), an exercise set, and a summary. Work top to bottom — each chapter builds on the ones before it.
